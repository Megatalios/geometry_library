# Геометрический движок

Легковесная C++ библиотека для вычислительной геометрии, параметрического моделирования и решения систем нелинейных уравнений. Движок написан с нуля без использования тяжеловесных внешних зависимостей, основаня цель - более углубленное понимание матричной алгебры, численных методов и архитектурной оптимизации.

## Ключевые возможности

### 1. Математическое ядро и численные методы
* **Многомерный метод Ньютона:** Реализован итеративный решатель для нахождения проекции точки на параметрические 3D-поверхности. В решателе реализованы **алгоритм возврата** и **градиентный спуск** для улучшения сходимости и надежности
* **Решение СЛАУ:** Аналитическое обращение матрицы Якоби 2x2 методом Крамера на каждой итерации с защитой от вырождения (деления на ноль).

### 2. Вычислительная геометрия
* **Параметрические кривые:** Отрезки прямой, кривые Безье и B-сплайны произвольной степени. Все реализовано в параметрических формулах
* **Алгоритм Кокса-де Бура:** Рекурсивное вычисление базисных функций для B-сплайнов с возможностью автоматической генерации узлового вектора.
* **Тензорное произведение:** Математический аппарат для построения 3D-поверхностей на основе сетки контрольных точек.
* **Дифференциальная геометрия:** Аналитическое вычисление первых, вторых и смешанных частных производных, а также векторов нормали.

### 3. Пространственная оптимизация
* **AABB (Axis-Aligned Bounding Box):** Пространственное кэширование.
* **Отсечение:** Быстрая проверка коллизий $O(1)$ до запуска метода Ньютона.

## Технологический стек
* **Язык:** C++ (С поддержкой стандарта C++14/17)
* **Сборка:** CMake
* **Архитектура:** ООП (интерфейсы `ICurve`, `ISurface`), композиция кэша, сокрытие данных от решателей.

## Сборка проекта

Проект использует CMake для кроссплатформенной сборки.

```bash
mkdir build
cd build
cmake ..
cmake --build .
```
